Walkthrough (2022)

My first project made with Unity, I made a map that resembles an Ikea store. It's more of an art game rather than an actual game because of the lack of mechanics.
About
You are in a closed Ikea store in an endless parking lot. Your only task is to find Mr. Piggy.

Play game on itch.io: https://soccyx.itch.io/walk-through

Applications
Made using Unity game engine and Maya modeling software.
