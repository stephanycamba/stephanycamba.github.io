The House Always Spins

About
A 3D first-person shooter made with a team of five people. This game was created in 1 month for Boss Rush Game Jam 2025 under the theme "Spin". I made the character and the animations.

Play game on itch.io:https://leyza.itch.io/the-house-always-spins

Applications
Maya and Unity

Combined my animations (Idle, walking, and basic attacks) in Unity using blend trees to test if each animation transitions smoothly. It starts with the idle, then walking + basic attacks.
